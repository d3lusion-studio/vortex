Vortex là một **game engine C++20 hướng dữ liệu, code-first**. Không có editor, không có định dạng
project độc quyền: game của bạn là một chương trình C++ link tới một thư viện.

## Hiện đã có gì

- **ECS** — registry sparse-set với component lưu đóng gói liên tục. System là hàm thuần.
- **Renderer** — sprite batching, sprite sheet và animation, tilemap import từ Tiled, particle, text
  bitmap và một lớp UI immediate-mode. Nhánh 3D cũng đã chạy: PBR GGX, shadow map PCF, HDR bloom với
  tonemap ACES.
- **RHI** — lớp trừu tượng backend với hai hiện thực: Vulkan và WebGPU (wgpu-native). Không một dòng
  code nào phía trên lớp này nhắc tới tên một graphics API.
- **Physics** — rigid body 2D, collider, joint, raycast và sensor thông qua Box2D.
- **Platform** — cửa sổ và input qua GLFW, cùng hệ thống action đặt tên phía trên.

## Vortex *không* phải là gì

- **Không phải engine xoay quanh editor.** Nếu bạn muốn kéo thả prefab vào viewport, hãy dùng Godot
  hoặc Unity. Vortex gần với Bevy hơn: bạn viết scene bằng code.
- **Chưa ổn định.** API còn thay đổi giữa các bản phát hành. Hãy ghim version.
- **Không phải engine 3D-first.** 3D chạy được, nhưng 2D mới là phần hoàn chỉnh.
