## Yêu cầu

| Công cụ | Tối thiểu | Ghi chú |
| --- | --- | --- |
| CMake | 3.23 | Presets v6 |
| Ninja | bất kỳ | Generator mặc định |
| Trình biên dịch C++ | GCC 13 / Clang 17 / MSVC 19.38 | C++20 |
| Vulkan SDK | 1.3 | Chỉ cần cho backend Vulkan |

## Build

```bash
git clone https://github.com/d3lusion-studio/vortex
cd vortex
cmake --preset release
cmake --build --preset release
```

Rồi chạy một ví dụ:

```bash
./build/release/examples/sprites
```

<Callout type="info">
  Backend renderer được chọn lúc chạy qua biến `VORTEX_RHI_API` (`vulkan` hoặc `webgpu`), nếu không
  đặt thì lấy giá trị cache `VORTEX_RHI_BACKEND` của CMake. Trên Wayland, các ví dụ hiện tạm ép dùng
  X11/XWayland cho tới khi swapchain Vulkan native hoàn thiện.
</Callout>

## Scene đầu tiên

`App` sở hữu cửa sổ, device, registry và vòng lặp. Bạn chỉ cần *hỏi* nó một hệ thống con, hệ thống đó
sẽ được tạo lười (lazy) cho scene đang hoạt động — không phải tự dựng `PhysicsWorld`, cũng không phải
nhớ gọi `step()`.

```cpp
#include "vortex/app/app.hpp"
#include "vortex/physics/components.hpp"

using namespace vortex;

int main() {
    app::App app{{.title = "Game của tôi", .width = 1280, .height = 720}};

    const ecs::Entity player = app.scene().spawn();
    app.registry().get<ecs::Transform2D>(player).position = {0.0f, 4.0f};
    app.registry().emplace<ecs::SpriteComp>(player, ecs::SpriteComp{
        .texture = app.whiteTexture(),
        .color   = Color::white(),
        .size    = {1.0f, 1.0f},
    });
    app.registry().emplace<physics::RigidBody2D>(player, physics::RigidBody2D{
        .restitution = 0.4f,
    });
    app.registry().emplace<physics::BoxCollider2D>(player, physics::BoxCollider2D{
        .halfExtents = {0.5f, 0.5f},
    });

    return app.run();
}
```

Link tới target của engine:

```cmake
find_package(Vortex REQUIRED)
add_executable(my_game main.cpp)
target_link_libraries(my_game PRIVATE Vortex::vortex)
```
